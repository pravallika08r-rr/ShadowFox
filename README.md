# Image Tagging Project

## Overview
This project uses TensorFlow and Convolutional Neural Networks (CNN) to classify images into:

- Cat
- Dog
- Car

## Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib
- VS Code

## Features
- Image classification using CNN
- Predicts image category
- Displays prediction confidence
- Training accuracy visualization

## Run Training
python train_model.py

## Run Prediction
python predict.py

## Example Output
Predicted Class: dog
Confidence: 97.45%
